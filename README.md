## Setup
### Prerequisites ###

* NodeJS = 10.15.3
* MariaDB = 10.4

### Install

1. Install `npm`
2. Execute `npm install`

### Configuration

1. Create `.env` file and fill in values, the format is as same as in `.env.example`

## Development and Deployment Scripts

### Development
* Start a offline server to test endpoints with localhost:3000
    ```
    node app.js
    ```
